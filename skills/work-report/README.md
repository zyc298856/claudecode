# 工作报告生成器 (Work Report Generator)

将散乱的工作记录整理为结构化的日报或周报。

## 功能特点

- **自动识别报告类型**：根据关键词自动判断生成日报还是周报，也可手动指定
- **智能分类**：按项目/模块将散乱事项自动分组归整
- **量化表达**：将定性描述转为定量表达，保留原始数据，不编造数据
- **中英双语**：根据输入语言自动匹配输出语言
- **多轮交互**：支持在生成后追问、补充、修改报告内容

## 使用方式

### 触发方式

- 说"写周报"、"整理日报"、"帮我写工作汇报"、"生成本周报告"
- 粘贴散乱的工作内容要求整理
- 说"weekly report"、"daily report"、"work summary"

### 示例

**输入**（散乱文本）：
```
周一开了产品会，确定了搜索优化的方案
周二到周三在写搜索模块的代码，基本完成了
周四测了一下发现几个bug，修了
还跟运营对了一下分享功能的需求
这周还帮同事看了一下性能问题，接口响应从2秒优化到800ms
```

**输出**：结构化周报，按项目分组，包含量化成果和下周计划。
