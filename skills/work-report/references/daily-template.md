# 工作日报模板

本文档提供日报的输出模板和示例，供 AI 生成报告时参考。

## 模板

```markdown
# 工作日报：{YYYY年MM月DD日}

**姓名**：{姓名}
**部门**：{部门名称}

---

## 今日完成

- {动词开头的事项描述}（{项目/模块}）
- {动词开头的事项描述}（{项目/模块}）
- ...

## 遇到问题

- {问题描述及当前状态}（如无问题可省略此节）

## 明日计划

- {动词开头的计划事项}
- {动词开头的计划事项}
- ...

---

> 本报告由 AI 辅助整理。
```

## 示例

以下是一个完整的日报示例：

```markdown
# 工作日报：2026年4月1日

**姓名**：丁雨薇
**部门**：AI STORE 平台开发组

---

## 今日完成

- 完成 work-report skill 的 SKILL.md 编写（work-report skill）
- 实现 format.py 中的报告类型自动识别功能（work-report skill）
- 修复 meeting-minutes skill 中 VTT 解析的边界问题（meeting-minutes skill）
- 参加 Q2 项目规划会议，整理会议纪要（项目管理）

## 遇到问题

- work-report 的周报模板在数据量较大时输出格式不稳定，需进一步优化

## 明日计划

- 完成 work-report skill 的 references 文档编写
- 开始第三个 skill 的需求调研
- 提交本周代码 review

---

> 本报告由 AI 辅助整理。
```

## 格式要点

1. **今日完成**：必填，每条以动词开头，标注所属项目/模块
2. **遇到问题**：可选，无问题时省略整个章节
3. **明日计划**：必填，如无明确计划则填写"待规划"
4. **事项粒度**：每条对应半天以内可完成的工作量
