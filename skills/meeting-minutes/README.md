# 会议纪要生成器 (Meeting Minutes Generator)

将会议录音转写文本或纯文本会议记录整理为结构化的 Markdown 会议纪要。

## 功能特点

- **多格式输入**：支持纯文本、SRT 字幕、VTT 字幕、微信聊天记录四种格式自动识别
- **智能提取**：自动提取会议标题、参会人、议题、讨论要点、决议和待办事项
- **结构化输出**：生成格式统一的 Markdown 会议纪要，包含待办事项表格
- **中英双语**：根据输入语言自动匹配输出语言
- **多轮交互**：支持在生成后追问、补充、修改纪要内容

## 使用方式

### 触发方式

- 说"整理会议记录"、"生成会议纪要"、"总结会议"
- 粘贴 SRT/VTT 字幕文件或纯文本会议内容
- 粘贴 Zoom/Teams/飞书/钉钉转写文本
- 粘贴微信群聊记录

### 示例

**输入**（纯文本）：
```
今天下午2点开了产品规划会，张明说搜索优化是最高优先级，占用户反馈35%。李华估计要2周开发1周测试。王芳说UI可以同步改。赵强希望先上分享功能...
```

**输出**：结构化 Markdown 会议纪要，包含议题分组、讨论要点、决议、待办事项表格。
