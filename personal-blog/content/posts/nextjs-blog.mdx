## 为什么选择 Next.js

Next.js 提供了以下优势：

1. **静态生成 (SSG)** — 博客页面在构建时生成，加载极快
2. **App Router** — 基于 React Server Components 的新路由系统
3. **MDX 支持** — 可以在 Markdown 中使用 React 组件

## 项目结构

```bash
app/
  posts/
    [slug]/page.tsx
  layout.tsx
lib/
  posts.ts
content/
  posts/*.mdx
```
